# NGM Service Overview

## Purpose

Nepal Government Modernization (NGM) provides transparent access to Nepal's judicial data, focusing on anti-corruption efforts and judicial modernization.

## Requirements

### Case Search
- Users can search court cases across Supreme, High, District, and Special Courts
- Users can filter cases by case number, case type, date range, court, plaintiff, and defendant
- Users can view case details including parties, hearing dates, judges, and current status
- Users can search in both Nepali and English
- Users can filter specifically for corruption-related cases

### CIAA Annual Reports
- Users can browse CIAA (Commission for Investigation of Abuse of Authority) annual reports
- Users can download report PDFs
- Users can view report metadata (year, publication date, title)
- Users can search reports by year or keyword

### Kanun Patrika (Nepal Law Journal)
- Users can browse Nepal's official legal gazette by year, volume, and issue
- Users can download Kanun Patrika PDFs
- Users can search for specific legal precedents or topics

### Data Coverage
- System covers Supreme Court (final appellate and constitutional matters)
- System covers 18 High Courts (appellate jurisdiction)
- System covers 77 District Courts (original jurisdiction)
- System covers Special Court (anti-corruption and financial crimes)
- System displays case numbers, registration dates, parties, hearing schedules, judges, and case status

### User Experience
- Platform is accessible to citizens, journalists, and researchers
- Interface supports both Nepali and English languages
- Design is responsive for mobile and desktop devices
- Navigation is intuitive for non-technical users
